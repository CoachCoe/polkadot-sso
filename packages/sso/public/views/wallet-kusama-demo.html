üîç API Endpoints

These are the new wallet-based endpoints that replace the hardcoded seed approach:

New Wallet-Based Endpoints:

  • POST /api/wallet-kusama/store - Store credential with wallet auth
  • POST /api/wallet-kusama/retrieve - Retrieve credential with wallet auth
  • GET /api/wallet-kusama/list - List credentials with wallet auth
  • GET /api/wallet-kusama/cost-estimate - Estimate storage cost
  • GET /api/wallet-kusama/health - Check network health

Old Hardcoded Endpoints (for comparison):

  • POST /api/kusama/store - Uses hardcoded seed (deprecated)
  • POST /api/kusama/retrieve - Uses hardcoded seed (deprecated)
